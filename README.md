## 만다라트 계획표란 ?

- 일본의 경영 전문가 마츠무라 야스오가 목표 달성을 위해 만든 기법을 활용한 계획표
- 가장 큰 목표를 세우고 이에 대한 해결점, 아이디어, 생각들을 Who, Why, What, Where, When을 통해 확산해 나가는 형태이다.
- 미국 메이저 리그에서 활약 중인 오타니 쇼헤이 선수가 학창 시절 때 작성한 계획표로 유명
